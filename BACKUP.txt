# backup.md

## Restore från backup i MySQL Workbench

1. Öppna `Server -> Data Import`
2. Välj `Import from Self-Contained File`
3. Välj filen `bokhandel_backup.sql`
4. Välj eller skapa `Default Target Schema`
5. Klicka `Start Import`
